How It Works

Under the hood, the TSA game is a small but complete game loop:

  • Unity C# scripts handle player movement, collisions, and game rules.
  • A simple state machine controls menus, in-game, pause, and game-over screens.
  • Health and score systems give players feedback and a reason to keep trying.
  • UI elements (tutorial text, buttons, labels) are wired to game events so the flow feels clean.
  • Camera and scene setup make sure the level is readable to someone seeing it for the first time.
